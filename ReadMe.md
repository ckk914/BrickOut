🎯 기능 요구사항<br>
 ▪숫자 박스 클릭 시 박스가 제거된다.<br>
 ▪제거될때 재미를 위해 애니메이션 추가<br>
 ▪숫자 순서가 틀렸을때 애니메이션 추가<br>
 ▪화면 전환 애니메이션 추가<br>
 ▪숫자 초기 25까지 표시.<br>
 ▪26~30은 1~5개 박스 사라질때 5개 추가 생성<br>
 ▪30까지만 생성하고 이후엔 화면에서 박스 안보이게 처리<br>
 ▪게임이 클리어 되면 클리어 화면을 보여준다.<br>
 ▪제한시간 내 게임을 클리어 못하면, 실패화면을 보여준다.<br>
